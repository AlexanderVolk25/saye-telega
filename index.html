🚀 Vless прокси

Подходит для любых приложений. Отсчёт времени начинается при первом подключении.

Как подключить:

  1. Скопируйте Vless ссылку
  2. Откройте приложение (v2rayNG, Nekoray, Shadowrocket)
  3. Нажмите «Импорт из буфера»
  4. Включите подключение

📡 MTProto прокси

Только для Telegram.

Как подключить:

  1. Скопируйте MTProto ссылку
  2. Telegram → Настройки → Прокси
  3. Добавить прокси → Вставить ссылку
  4. Сохранить
